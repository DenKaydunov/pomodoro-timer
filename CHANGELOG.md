# История изменений

## Версия 2.0.0 (Апрель 2026)

### Крупные изменения

- ♻️ **Полная переработка проекта** на Node.js и Tailwind CSS
- 📦 Добавлена система сборки с npm скриптами
- 🎨 Переход с кастомного CSS на Tailwind utility classes
- 📁 Разделение на модульную структуру (src/ и dist/)

### Технологический стек

**Добавлено:**
- Node.js для управления зависимостями
- Tailwind CSS v3 для стилизации
- PostCSS и Autoprefixer
- npm скрипты для сборки и разработки

**Сохранено:**
- Vanilla JavaScript (без фреймворков)
- Web Audio API для звуков
- Notifications API для уведомлений
- Все функции оригинального приложения

### Новая структура проекта

```
pomodoro-timer/
├── src/                    # Исходные файлы
│   ├── input.css          # Tailwind директивы
│   └── app.js             # JavaScript логика
├── dist/                   # Собранное приложение
│   ├── index.html         # HTML с Tailwind классами
│   ├── output.css         # Сгенерированный CSS
│   └── app.js             # Скопированный JS
├── tailwind.config.js     # Конфигурация Tailwind
├── package.json           # Зависимости и скрипты
├── CLAUDE.md              # Руководство для Claude Code
├── QUICKSTART.md          # Быстрый старт
└── index.html.old         # Старая версия (single-file)
```

### Команды

- `npm install` - установка зависимостей
- `npm run build` - полная сборка проекта
- `npm run dev` - режим разработки с watch
- `npm run build:css` - только сборка CSS
- `npm run copy:js` - только копирование JS

### Миграция

Для использования новой версии:

1. Установите Node.js (если еще не установлен)
2. Запустите `npm install`
3. Соберите проект: `npm run build`
4. Откройте `dist/index.html` в браузере

Старая версия сохранена в `index.html.old` и может быть использована без зависимостей.

### Преимущества новой версии

✅ Модульная архитектура
✅ Современный workflow разработки
✅ Меньший размер CSS (минификация)
✅ Легче поддерживать и расширять
✅ Стандартизованные стили через Tailwind
✅ Автоматическая пересборка в dev режиме

## Версия 1.0.0 (Первоначальный релиз)

- 🎉 Первый релиз Pomodoro Timer
- Single-file приложение (HTML + CSS + JS в одном файле)
- Базовая функциональность Pomodoro техники
- Три режима: работа, короткий и длинный перерыв

- Счетчик завершенных помодоро
