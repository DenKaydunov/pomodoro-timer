# 🍅 Помодоро Таймер

Простое веб-приложение для управления временем по технике Pomodoro.

## Что такое техника Помодоро?

Техника Pomodoro - это метод управления временем, который помогает сосредоточиться на работе:
1. Работайте 25 минут (один "помодоро")
2. Отдыхайте 5 минут (короткий перерыв)
3. После 4 помодоро сделайте длинный перерыв (15 минут)

## Как запустить

1. Откройте файл `index.html` в любом современном браузере (Chrome, Firefox, Safari, Edge)
2. Разрешите уведомления, если хотите получать оповещения о завершении таймера

## Как использовать

### Основные функции:

- **Старт/Пауза** - нажмите кнопку "Старт" или клавишу `Пробел` для запуска/паузы таймера
- **Сброс** - нажмите кнопку "Сброс" или клавишу `R` для сброса текущего таймера
- **Переключение режимов** - выберите один из трех режимов:
  - **Работа** - 25 минут для концентрированной работы
  - **Короткий перерыв** - 5 минут отдыха
  - **Длинный перерыв** - 15 минут отдыха

### Автоматические функции:

- ✅ Таймер автоматически переключается на перерыв после завершения работы
- ✅ После 4 помодоро автоматически включается длинный перерыв
- ✅ Звуковой сигнал при завершении таймера
- ✅ Браузерные уведомления (требуется разрешение)
- ✅ Счетчик завершенных помодоро

### Горячие клавиши:

- `Пробел` - Старт/Пауза
- `R` - Сброс таймера

## Особенности

- 🎨 Красивый современный дизайн с цветовой индикацией режимов
- 📊 Прогресс-бар для визуального отображения оставшегося времени
- 📱 Адаптивный дизайн - работает на компьютерах, планшетах и смартфонах
- 🔔 Уведомления в браузере
- 🎵 Звуковые оповещения
- 📈 Подсчет завершенных помодоро-сессий

## Цветовая схема

- 🔴 **Красный фон** - режим работы (помодоро)
- 🟢 **Зеленый фон** - короткий перерыв
- 🔵 **Синий фон** - длинный перерыв

## Технические детали

Приложение создано с использованием:
- HTML5
- CSS3 (с адаптивным дизайном)
- Vanilla JavaScript (без фреймворков)

### Используемые браузерные API:
- Web Audio API (для звуковых сигналов)
- Notifications API (для уведомлений)
- setInterval (для таймера)

## Советы по использованию

1. **Планируйте задачи**: перед началом работы определите, что хотите сделать за один помодоро
2. **Не отвлекайтесь**: во время работы концентрируйтесь только на одной задаче
3. **Используйте перерывы**: во время перерыва встаньте, потянитесь, отдохните от экрана
4. **Отслеживайте прогресс**: счетчик помодоро поможет понять, сколько вы работали
5. **Настройте уведомления**: разрешите браузерные уведомления, чтобы не пропустить завершение таймера
